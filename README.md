## Overview
This project is an outdoor autonomous delivery robot that navigates
using GPS coordinates and avoids obstacles in real-time.

## Features
- GPS based navigation
- Obstacle detection
- Autonomous movement
- Target coordinate tracking
- Expandable to IoT tracking

## Working
1. GPS reads current latitude & longitude.
2. Robot calculates distance to target.
3. Moves forward until target reached.
4. Avoids obstacles using ultrasonic sensor.

## Applications
- Smart campus delivery
- Warehouse robotics
- Smart city automation
- Defense and rescue robots

## Future Enhancements
- Mobile app tracking
- Live Google Maps integration
- Camera surveillance
- AI path optimization
